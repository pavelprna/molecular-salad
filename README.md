# Molecular Salad

Приложение для компоновки своих салатов или выбора готовых.

Можно выбрать готовый салата из списка доступных (список салатов получаем из данных API)

Есть возможность составить свой собственный салат из доступных молекул на складе.

Наконец, можно создать заказ с готовым или составленным салатом.

## Запуск проекта

`yarn yarn start`
